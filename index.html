The Effect of Latitude on Weather Trends

An analysis was started on how weather is affected by latitude. A number of variables were outlined within the analysis including: temperature, humidity, cloud cover, and wind speed.

The initial hypothesis, as latitude approaches 0 degrees, average temperature increases. To begin, queries were created to probe the Open Weather API in order to obtain weather data for a given time period (one day).

After importing the data into Jupyter Notebook, a Python code was constructed to take the data and plot that information onto a number or graphically representations using the MatPlotLib functionality.
